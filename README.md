### Operator

```
myVar=6 
myVar+=2  -> 8
myVar-=2  -> 4
myVar*=2  -> 12
myVar/=2  -> 3
myVar//=8 -> 0 (floor -> 整除)
myVar%=2  -> 0 (remainder)
myVar**=2 -> 6^2 = 36 -> assign power 2 to a
myVar**=3 -> 6^3 = 216 -> assign power 3 to a
```
